# 🧠 Human Body Detection using Computer Vision

> A computer vision project focused on detecting and analyzing human body structures using real-time video input.

---

## 📌 About

This project focuses on building a **human body detection system using computer vision**.

It is currently in its **early phase**, with the aim of developing a system that can assist in:

- **Medical anatomical identification**
- **Human body detection and analysis**
- **Future diagnostic support systems**

The goal is to explore how computer vision can be applied to healthcare-related use cases in a simple and practical way.

---

## 🎯 Objective

- Detect human body structures using a webcam  
- Analyze basic body movement and positioning  
- Build a foundation for future medical diagnostic support  

---

## 🛠️ Tech Stack (Planned)

- Python  
- OpenCV  
- MediaPipe  
- Deep Learning (future scope)

## 🚀 Current Status

- 🔹 Accuracy and performance improvements are in progress
